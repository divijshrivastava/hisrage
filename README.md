# HisRage - Men's Jewellery E-Commerce Website

A bold, aggressive men's jewellery website designed to embody masculine rage and power. Built with pure HTML, CSS, and JavaScript.

## Brand Concept

**HisRage** focuses on the raw, untamed power of masculinity. The brand channels primal rage and warrior spirit into premium 316L stainless steel jewellery pieces.

### Brand Messaging
- "Unleash the Rage" - Main tagline
- "Steel Forged in Fire. Built for Warriors"
- "Channel Your Inner Beast"
- Focus on power, dominance, and primal masculinity

## Website Structure

### Pages/Sections
1. **Hero Section** - Full-screen impact with dramatic imagery
2. **Product Categories** - Rings, Bracelets, Chains & Pendants
3. **Features** - 6-Month Warranty, Quick Shipping, Premium Steel
4. **About** - Brand story focusing on rage and warrior mentality
5. **Lifestyle** - Powerful imagery showcasing the brand aesthetic
6. **Warranty** - Trust-building section
7. **Contact** - Customer engagement ("Join the Pack")

## Features

### Design Elements
- **Dark Theme**: Black backgrounds with gold and red accents
- **Rage Colors**:
  - Primary Gold: #d4af37
  - Rage Red: #cc0000
  - Fire Orange: #ff6600
- **Bold Typography**: Playfair Display for headings, Inter for body
- **Aggressive Imagery**: Fire, warriors, animals (tiger, lion)

### Technical Features
- Responsive design (mobile, tablet, desktop)
- Smooth scrolling navigation
- Parallax effects on hero section
- Custom cursor effect (desktop only)
- Scroll-triggered animations
- Interactive hover effects
- Form validation

### Interactive Elements
- Auto-hiding navbar on scroll down
- Category cards with hover overlays
- Animated statistics counters
- Product showcase with smooth transitions
- Mobile menu toggle

## How to Use

### Viewing the Website
1. Open `index.html` in any modern web browser
2. Navigate using the top menu or scroll through sections
3. Best viewed on desktop for full effects

### Customization
- **Colors**: Edit CSS variables in `styles.css` (lines 8-18)
- **Content**: Update text in `index.html`
- **Images**: Replace images in `/images` folder
- **Branding**: Search and replace "HISRAGE" throughout files
